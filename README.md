1️. What is the difference between var, let, and const?

Ans: var holo function-scoped and re-declare kora jay, let holo block-scoped and re-declare kora jay na but update kora jay, and const holo block-scoped and once value assign korle ar change ba re-declare kora jay na.

 2. What is the spread operator (...)?

Ans: Spread operator (...) use kora hoy array/object er elements gula ke expand ba copy korar jonno easily.

3. What is the difference between map(), filter(), and forEach()?

Ans: map() returns a new array with modified data, filter() returns a new array based on condition, and forEach() just loop chalay but kono new array return kore na.

4. What is an arrow function?

Ans: An arrow function is a shorter way to write a function in JavaScript using =>, and it does not have its own this context.

5. What are template literals?

Ans: Template literals are a modern way to create strings in JavaScript using **backticks ()**, and you can embed variables or expressions inside ${}`.
